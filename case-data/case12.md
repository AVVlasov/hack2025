**1. Концепция проекта**

```
Описание кейса / проблемы

ИТ-академия сталкивается с комплексом проблем в образовательном процессе:

1. Менеджеры тратят до 40% времени на ответы 
на типовые вопросы потенциальных клиентов: 
"Какой курс мне подойдет?", "Чем отличается 
Data Analyst от Data Scientist?", "Какие 
вакансии после курса?".

2. После корпоративного обучения собираются 
фидбэки, но их анализ занимает много времени. 
Теряются ценные инсайты: какие темы были 
полезными, где пробелы, что нужно улучшить 
в программах.

3. После завершения обучения связь с 
корпоративным клиентом часто прерывается. 
Отсутствует проактивный подход к продаже 
следующих курсов и долгосрочному 
сопровождению клиентов.

Необходима комплексная система на базе ИИ, 
которая автоматизирует взаимодействие с 
клиентами на всех этапах: от первого 
касания до удержания и повторных продаж.

Описание образа результата

Комплексная ИИ-система для ИТ-академии, 
которая работает на трех ключевых этапах:

1. AI-Консультант на сайте отвечает на 
вопросы потенциальных клиентов 24/7, 
подбирает курсы, формирует лиды.

2. AI-Аналитик автоматически обрабатывает 
фидбэки после обучения, выявляет проблемные 
зоны и генерирует рекомендации по улучшению 
программ.

3. AI-Менеджер по удержанию поддерживает 
связь с корпоративными клиентами, 
отслеживает их развитие и предлагает 
следующие образовательные продукты.

Результат: снижение нагрузки на менеджеров 
на 40%, повышение конверсии лидов на 25%, 
увеличение повторных продаж на 30%.

Целевая аудитория

Внутренние пользователи:
- Менеджеры по продажам и обучению
- Методисты и руководители программ
- Преподаватели
- Администраторы курсов

Внешние пользователи:
- Потенциальные клиенты (B2C и B2B)
- Корпоративные заказчики
- HR-менеджеры компаний-клиентов
- Выпускники программ
```

**2. Цели и задачи**

```
Бизнес-цели

- Сократить время менеджеров на обработку 
входящих запросов с 40% до 15%
- Увеличить конверсию из лида в клиента 
на 25%
- Повысить скорость анализа фидбэков с 
3-5 дней до 30 минут
- Увеличить повторные продажи корпоративным 
клиентам на 30%
- Сократить отток корпоративных клиентов 
с 35% до 20%
- Повысить NPS (Net Promoter Score) программ 
с 7 до 9 баллов
- Ускорить адаптацию учебных программ под 
рынок в 3 раза

Пользовательские цели:

Потенциальный клиент:
- Получить быструю консультацию по курсам 
в любое время
- Понять, какая программа подходит под 
цели и опыт
- Узнать перспективы трудоустройства

Менеджер по продажам:
- Получать квалифицированные лиды с 
заполненной анкетой
- Экономить время на типовых вопросах
- Фокусироваться на сложных запросах

Методист/Руководитель программы:
- Быстро получать структурированную 
аналитику по фидбэкам
- Видеть конкретные точки для улучшения 
программ
- Отслеживать динамику удовлетворенности

Корпоративный клиент:
- Получать персонализированные предложения 
по развитию команды
- Видеть релевантные рекомендации новых 
курсов
- Чувствовать долгосрочную поддержку
```

**AI-Консультант (первая линия поддержки):**
```
- Web-виджет на сайте edu.itpark.tech
- База знаний: описания всех курсов, 
требования, стоимость, расписание
- Классификация запросов (выбор курса, 
стоимость, расписание, карьера)
- Задает уточняющие вопросы: опыт, цели, 
бюджет, формат обучения
- Рекомендует 2-3 курса с обоснованием
- Показывает актуальные вакансии по 
направлению (интеграция с hh.ru)
- Форма захвата лида: имя, контакты, 
выбранный курс
- Передача сложных запросов менеджеру
- Интеграция с CRM для записи лидов

**AI-Аналитик отзывов:**
- Интеграция с Яндекс.Формами и email
- Автоматический сбор текстовых отзывов 
после завершения курса
- Тематическое моделирование (Topic Modeling): 
выделение основных тем в отзывах
- Анализ тональности (Sentiment Analysis) 
по каждой теме
- Выявление проблемных зон: "сложность", 
"темп", "нехватка практики", "качество 
материалов"
- Генерация сводки по преподавателям с 
рекомендациями
- Еженедельный отчет для руководителей 
программ
- Экспорт в PDF и Excel

**AI-Менеджер по удержанию:**
- Триггерная email-рассылка после 
завершения обучения
- Через 1 месяц: сбор обратной связи о 
применении навыков
- Через 3 месяца: опрос о карьерных 
изменениях и новых потребностях
- Мониторинг новостей компаний-клиентов 
(интеграция с новостными API)
- Определение триггеров для предложения 
курсов: новый продукт, открытие отдела, 
найм сотрудников
- Персонализированные рекомендации: 
следующий курс на основе пройденных программ
- Автоматическая генерация черновиков КП 
для менеджера
- Уведомления менеджерам о горячих лидах

**Dashboard для менеджеров и руководителей:**
- Статистика по обращениям AI-Консультанта
- Топ-3 запроса недели
- Конверсия лидов по источникам
- Аналитика отзывов: средний NPS, динамика 
по курсам
- Список клиентов для реактивации
- Прогноз повторных продаж
