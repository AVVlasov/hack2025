1. **Концепция проекта**

```
Описание кейса / проблемы Необходимо разработать систему 
интеллектуального скоринга стартапов, 
которая будет быстро определять 
жизнеспособность стартапа без глубокого 
анализа. Текущее время анализа: 2-3 дня, 
цель: 15-30 минут. В 40% стартапов 
скрытые проблемы пропускаются при 
ручной проверке. Каждый аналитик 
оценивает субъективно, отсутствует 
единая методология. Система должна 
анализировать данные из множества 
источников: питчи, финансы, социальные 
сети, GitHub, новости, судебные реестры.

Описание образа результата
Венчурный инвестор загружает питч-
колоду стартапа и за 15 минут получает 
детальный скоринг-отчёт (4-5 страниц) с 
оценкой 0-100 баллов, breakdown по 8 
категориям (команда, рынок, продукт, 
финансы), 5-7 ключевыми риск-
факторами и рекомендациями по их 
проверке. Предсказание модели 
совпадает с реальностью на 80%+.

Целевая аудитория Венчурные фонды (средние и крупные, 
50+ сделок в год), корпоративные 
венчурники (R&D подразделения крупных 
компаний), бизнес-ангелы 
(независимые инвесторы), 
госагентства (АО "РВК", фонды 
развития, ФРИИ), 
коворкинги/акселераторы (программы 
поддержки стартапов).
```
**2. Цели и задачи**

```
Бизнес-цели Сократить время на первичный скрининг 
стартапов с 2-3 дней до 30 минут
Увеличить точность прогнозов успеха 
стартапов на 30-40%
Снизить долю неудачных инвестиций с 
25% до 10%
Обработать 1000+ стартапов в месяц 
для фильтрации пайплайна
Помочь инвесторам принять решение о 
финансировании на 50% быстрее

Пользовательские цели: Венчурный инвестор: получить 
объективный скоринг без предвзятости 
для быстрого отбора
Аналитик фонда: выявить скрытые риски 
и возможности, сэкономить время на 
рутинный анализ
Стартапер: понять, как улучшить проект 
и что инвесторы видят как слабые 
стороны
CEO фонда: снизить риски портфеля, 
получить данные для стратегических 
решений
```
